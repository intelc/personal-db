# personal_db

personal_db is a Python-based personal data layer that exposes user data (GitHub commits, Whoop, screen time, iMessage, and manual habits) to AI agents via an MCP server, providing a unified, queryable interface for personal quantified-self and productivity data.

- Design spec: [docs/superpowers/specs/2026-04-25-personal-db-v0-design.md](docs/superpowers/specs/2026-04-25-personal-db-v0-design.md)
- Implementation plan: [docs/superpowers/plans/2026-04-25-personal-db-v0.md](docs/superpowers/plans/2026-04-25-personal-db-v0.md)

For setup, see `scripts/install_dev.sh`.
